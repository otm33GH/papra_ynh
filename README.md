The `testing` branch can be tested using:
```
# fresh install:
sudo yunohost app install https://github.com/YunoHost-Apps/papra_ynh/tree/testing

# upgrade an existing install:
sudo yunohost app upgrade papra -u https://github.com/YunoHost-Apps/papra_ynh/tree/testing
```

You can also switch to the testing branch to update from testing by default (as same as for APT when you chose to use a testing repos) with this command:
```bash
sudo yunohost app setting papra upgrade_channel -v testing
```
